GeneratePress Implementation Notes

In the WordPress Customizer under GeneratePress ‚Üí Colors, map the palette to these fields:

Background Color ‚Üí #F7F3EC (Cloud White)  |  Text Color ‚Üí #2C2C2C (Ash Dark)

Link Color ‚Üí #1A9BB5 (Tropical Teal)  |  Link Color Hover ‚Üí #C84B0A (Volcanic Ember)

Navigation Background ‚Üí #5C3A1E (Driftwood Dark)  |  Navigation Text ‚Üí #EDD9A3 (Pale Sand)

Navigation Hover/Current ‚Üí #F5A623 (Torch Flame)  |  Footer Background ‚Üí #0A6E8A (Deep Lagoon)

Then paste the complete :root {} block above into Appearance ‚Üí Customize ‚Üí Additional CSS to make all variables globally available.
