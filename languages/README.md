# KEALOA Reference - Language Files

This directory contains translation files for the KEALOA Reference plugin.

## Creating Translations

1. Use a tool like Poedit to create translations
2. Load the `kealoa-reference.pot` template file
3. Translate strings to your target language
4. Save as `kealoa-reference-{locale}.po` (e.g., `kealoa-reference-es_ES.po`)
5. Compile to `kealoa-reference-{locale}.mo`

## Available Strings

All translatable strings in the plugin use the `kealoa-reference` text domain.
